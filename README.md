## :toolbox: Getting Started

### :gear: Installation

npm install
```bash
npm install
```


### :running: Run Locally

Clone the project

```bash
https://github.com/ksoni15/MindCanvas
```
Client side
```bash
npm run dev
```
Server side (Please check MongoDB connected)
```bash
npm start
```


### :triangular_flag_on_post: Deployment

Client side npm build
```bash
npm run build
```
